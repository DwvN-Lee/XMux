# Legacy Python Runtime

This archive preserves the Python XMux runtime scripts that were replaced by
the Node mailbox/runtime path.

The active runtime now uses:

- `dist/bin/xmux-mailbox.js`
- `src/mailbox/*`
- `scripts/*.js`

These Python files are retained for historical reference only. They should not
be imported, executed, or used by the active XMux CLI, MCP server, bridge, or
packaging paths.
